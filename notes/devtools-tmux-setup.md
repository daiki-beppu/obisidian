# tmux キーバインド一覧

## Prefix キーバインド

Prefix: `Ctrl-q`

### セッション管理

| キー | 説明 |
|------|------|
| `$` | セッション名を変更 |
| `(` | 前のセッションに切り替え |
| `)` | 次のセッションに切り替え |
| `d` | セッションからデタッチ |
| `s` | セッション一覧から選択 |
| `D` | クライアント一覧から選択 |
| `C-z` | クライアントを一時停止 |

### ウィンドウ管理

| キー | 説明 |
|------|------|
| `c` | 新しいウィンドウを作成 |
| `&` | 現在のウィンドウを削除（確認あり） |
| `,` | ウィンドウ名を変更 |
| `.` | ウィンドウを移動 |
| `0-9` | 指定番号のウィンドウを選択 |
| `'` | インデックスを指定してウィンドウを選択 |
| `n` | 次のウィンドウへ |
| `p` | 前のウィンドウへ |
| `l` | 最後に使用したウィンドウへ |
| `w` | ウィンドウ一覧から選択 |
| `f` | ウィンドウを検索 |
| `<` | ウィンドウを前に入れ替え（リピート可） |
| `>` | ウィンドウを後ろに入れ替え（リピート可） |
| `C-n` | 次のウィンドウへ |
| `C-p` | 前のウィンドウへ |
| `M-n` | 次の警告ありウィンドウへ |
| `M-p` | 前の警告ありウィンドウへ |

### ペイン管理

#### ペイン分割

| キー | 説明 |
|------|------|
| `"` | 縦に分割（カレントパス維持） |
| `%` | 横に分割（カレントパス維持） |
| `-` | 縦に分割（カレントパス維持） |
| `\|` | 横に分割（カレントパス維持） |
| `\` | フルサイズで横に分割（カレントパス維持） |
| `_` | フルサイズで縦に分割（カレントパス維持） |

#### ペイン選択・移動

| キー | 説明 |
|------|------|
| `o` | 次のペインを選択 |
| `;` | 最後に使用したペインへ |
| `h` | 左のペインを選択 |
| `j` | 下のペインを選択 |
| `k` | 上のペインを選択 |
| `l` | 右のペインを選択 |
| `C-h` | 左のペインを選択 |
| `C-j` | 下のペインを選択 |
| `C-k` | 上のペインを選択 |
| `C-l` | 右のペインを選択 |
| `Up/Down/Left/Right` | 矢印キーでペイン選択（リピート可） |
| `q` | ペイン番号を表示 |

#### ペイン操作

| キー | 説明 |
|------|------|
| `!` | ペインを新しいウィンドウとして分離 |
| `x` | 現在のペインを削除（確認あり） |
| `z` | ペインのズーム切り替え |
| `{` | ペインを上に入れ替え |
| `}` | ペインを下に入れ替え |
| `C-o` | ペインを回転 |
| `M-o` | ペインを逆回転 |
| `m` | ペインをマーク |
| `M` | ペインをマーク（モード設定） |

#### ペインリサイズ

| キー | 説明 |
|------|------|
| `H` | ペインを左に5セル拡大（リピート可） |
| `J` | ペインを下に5セル拡大（リピート可） |
| `K` | ペインを上に5セル拡大（リピート可） |
| `L` | ペインを右に5セル拡大（リピート可） |
| `M-Up/Down/Left/Right` | 矢印キーで5セルずつリサイズ（リピート可） |
| `C-Up/Down/Left/Right` | 矢印キーで1セルずつリサイズ（リピート可） |
| `S-Up/Down/Left/Right` | クライアント表示領域を10セル調整（リピート可） |

#### レイアウト

| キー | 説明 |
|------|------|
| `Space` | 次のレイアウトに切り替え |
| `M-1` | even-horizontal レイアウト |
| `M-2` | even-vertical レイアウト |
| `M-3` | main-horizontal レイアウト |
| `M-4` | main-vertical レイアウト |
| `M-5` | tiled レイアウト |
| `M-6` | main-horizontal-mirrored レイアウト |
| `M-7` | main-vertical-mirrored レイアウト |
| `E` | レイアウトを均等に調整 |

### コピーモード・バッファ

| キー | 説明 |
|------|------|
| `[` | コピーモードに入る |
| `]` | バッファをペースト |
| `#` | バッファ一覧を表示 |
| `=` | バッファ選択 |
| `PPage` | コピーモードに入り1ページ上へ |

### 設定・その他

| キー | 説明 |
|------|------|
| `:` | コマンドプロンプトを開く |
| `?` | キーバインド一覧を表示 |
| `/` | キーを指定してバインドを検索 |
| `t` | 時計を表示 |
| `i` | 現在のペイン情報を表示 |
| `~` | メッセージログを表示 |
| `r` | 設定ファイルを再読み込み |
| `R` | 設定ファイルを再読み込み（カスタム） |
| `C` | カスタマイズモードに入る |
| `DC` | クライアントをリフレッシュ（リピート可） |
| `C-q` | Prefixキーを送信 |

### プラグイン（TPM）

| キー | 説明 |
|------|------|
| `I` | プラグインをインストール |
| `U` | プラグインを更新 |
| `M-u` | 未使用プラグインを削除 |

### セッション保存・復元（tmux-resurrect）

| キー | 説明 |
|------|------|
| `C-s` | セッションを保存 |
| `C-r` | セッションを復元 |

## コピーモード（Emacs スタイル）

コピーモードに入るには: `Prefix + [`

### 基本操作

| キー | 説明 |
|------|------|
| `Escape` / `q` / `C-c` | コピーモードを終了 |
| `C-Space` | 選択開始 |
| `M-w` / `C-w` | 選択範囲をコピーして終了 |
| `C-k` | 行末までコピーして終了 |
| `C-g` | 選択をクリア |
| `X` | マークを設定 |
| `M-x` | マークにジャンプ |
| `P` | 位置表示の切り替え |

### カーソル移動

| キー | 説明 |
|------|------|
| `C-b` / `Left` | 左へ |
| `C-f` / `Right` | 右へ |
| `C-p` / `Up` | 上へ |
| `C-n` / `Down` | 下へ |
| `C-a` / `Home` | 行頭へ |
| `C-e` / `End` | 行末へ |
| `M-b` | 前の単語へ |
| `M-f` | 次の単語の末尾へ |
| `M-m` | インデントの先頭へ |
| `C-l` | カーソルを画面中央へ（垂直） |
| `M-l` | カーソルを画面中央へ（水平） |
| `M-r` | 画面中央行へ |
| `M-R` | 画面最上行へ |

### ページ・スクロール

| キー | 説明 |
|------|------|
| `Space` / `NPage` / `C-v` | 次のページ |
| `PPage` / `M-v` | 前のページ |
| `M-Up` | 半ページ上へ |
| `M-Down` | 半ページ下へ |
| `C-Up` | 1行スクロールアップ |
| `C-Down` | 1行スクロールダウン |
| `M-<` | 履歴の最初へ |
| `M->` | 履歴の最後へ |
| `r` | ペインから再読み込み |

### 検索

| キー | 説明 |
|------|------|
| `C-r` | インクリメンタル検索（上方向） |
| `C-s` | インクリメンタル検索（下方向） |
| `n` | 次の検索結果 |
| `N` | 前の検索結果 |

### ジャンプ

| キー | 説明 |
|------|------|
| `f` | 前方の文字にジャンプ |
| `F` | 後方の文字にジャンプ |
| `t` | 前方の文字の手前にジャンプ |
| `T` | 後方の文字の手前にジャンプ |
| `;` | ジャンプを繰り返す |
| `,` | ジャンプを逆方向に繰り返す |
| `g` | 指定行へ移動 |
| `C-M-b` | 前の対応括弧へ |
| `C-M-f` | 次の対応括弧へ |
| `M-{` | 前の段落へ |
| `M-}` | 次の段落へ |

### その他

| キー | 説明 |
|------|------|
| `R` | 矩形選択モード切り替え |
| `M-1` ~ `M-9` | コマンドリピート回数指定 |

### マウス操作

| 操作 | 説明 |
|------|------|
| `MouseDown1Pane` | ペイン選択 |
| `MouseDrag1Pane` | 選択開始 |
| `MouseDragEnd1Pane` | 選択範囲をコピー |
| `WheelUpPane` | 5行スクロールアップ |
| `WheelDownPane` | 5行スクロールダウン |
| `DoubleClick1Pane` | 単語選択してコピー |
| `TripleClick1Pane` | 行選択してコピー |

## 注記

- `リピート可` と記載されているキーは、一度押した後、Prefixなしで連続して実行できます
- デフォルトのPrefixキーは `Ctrl-b` です
- コピーモードは `setw -g mode-keys vi` または `setw -g mode-keys emacs` で切り替えられます
- 上記の一覧は現在の tmux 設定（カスタマイズ含む）に基づいています
