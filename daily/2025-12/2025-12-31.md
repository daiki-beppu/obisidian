### tmux のキーバインド

よく使いそうなやつをまとめる

prefix = control + q

- `prefix + !` : 新しいウィンドウを立ちあげる
- `prefix + &` : ウィンドウを削除
- `prefix + "` : ウィンドウを横分割
- `prefix + |` : ウィンドウを縦分割 

### calude code で改行できない問題

⏺ エラーの原因と対策

  ⚠️ エラーメッセージ

  [ERROR] Unable to load config "/Users/mba/.config/alacritty/alacritty.toml": Config error: TOML parse error at line 28, column 11

  28 | chars = "\x1b\r"
     |           ^
  missing escaped value, expected `b`, `f`, `n`, `r`, `\`, `"`, `u`, `U`

  See log at /var/folders/df/2dv2pvsx255_n6hg5wklcryc0000gn/T/Alacritty-13486.log ($ALACRITTY_LOG)

  🔍 原因

  Alacrittyの設定ファイル（alacritty.toml）28行目で TOMLの仕様に対応していないエスケープシーケンス を使用していました。

  chars = "\x1b\r"  # ❌ TOMLでは \x形式はサポートされていない

  TOMLファイルでは、16進数形式（\xHH）のエスケープシーケンスが使えません。エラーメッセージが示すように、TOMLでは b, f, n, r, \, ", u, U のみがエスケープ文字として認識されます。

  ✅ 対策

  エスケープシーケンスを Unicode形式（\uXXXX）に変更しました。

  chars = "\u001B\r"  # ✅ TOMLで正しく認識される形式

  📝 詳細

  - エスケープ文字: \x1b → \u001B （どちらもESCキー = ASCII 27）
  - 設定の意味: Shift+Returnキーで「ESC + 改行」を送信
  - TOML仕様: Unicode形式（\uXXXX）のみサポート

  参考: TOMLで使えるエスケープシーケンス

  - \u001B - Unicode（4桁）
  - \U0000001B - Unicode（8桁）
  - \n, \r, \t - 改行、復帰、タブ
  - \\, \" - バックスラッシュ、ダブルクォート
  - \b, \f - バックスペース、フォームフィード

  今後同様のエラーが出た場合は、\x 形式を \u 形式に置き換えることで解決できます。
