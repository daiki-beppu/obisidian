#ClaudeCode #コンピュータサイエンス #CS50 
**質問1：Claude CodeのChrome連携機能によって、ターミナルから直接行えるようになる主な作業は何ですか？**

• **正解と解説**: ブラウザの自動化により、**ライブデバッグ**（コンソールエラーやDOMの読み取り）、**デザイン検証**（Figmaとの比較）、**Webアプリテスト**、**データ抽出**などが可能になります。

**質問2：Chrome連携を利用するために必要なソフトウェアのバージョンやプランなどの前提条件は何ですか？**

• **正解と解説**: **Google Chrome**、**Claude in Chrome拡張機能 (v1.0.36以上)**、**Claude Code CLI (v2.0.73以上)**、および**有料Claudeプラン**（Pro, Team, Enterprise）が必要です。

**質問3：Chrome連携を有効にしてClaude Codeを起動するためのコマンドは何ですか？**

• **正解と解説**: `claude --chrome` です。また、`/chrome` コマンドで設定管理が可能です。

**質問4：Claude Code on the webにおいて、すべてのインターネット通信（アウトバウンドトラフィック）はどのような仕組みを経由して行われますか？**

• **ユーザーの回答：** わからない

• **正解と解説：** **セキュリティプロキシ (Security Proxy)** です。これを通じて、悪意のあるリクエストからの保護やコンテンツフィルタリングが行われます。

**質問5：アヒルのおもちゃなどの無生物にコードの論理を説明することでミスに気づくデバッグ手法を何と呼びますか？**

• **ユーザーの回答：** ラダーデバッキング

• **正解と解説：** 正しくは**ラバーダック・デバッグ**です。

**質問6：一つのGitリポジトリに対して、複数の作業ディレクトリを同時に保持できる機能を何と呼びますか？**

• **ユーザーの回答：** gitworktree

• **正解と解説：** **Git Worktree** です。異なるブランチを別々のフォルダに展開でき、AIエージェントによる並列作業に適しています。

**質問7：Slack連携の「ルーティングモード」のうち、メッセージを分析してコーディングタスクとそれ以外を自動で振り分けるモード名は何ですか？**

• **ユーザーの回答：** わからない

• **正解と解説：** **「Code + Chat」モード** です。

**質問8：C言語のコンパイルプロセスにおいて、ヘッダーファイルの内容をソースコードに展開する最初の段階を何と呼びますか？**

• **ユーザーの回答：** コンパイル

• **正解と解説：** 正しくは**プリプロセッシング**です。

**質問9：コンパイルプロセスの最後である「リンキング」において、バイナリファイルは何という1つのファイルに結合されますか？**

• **ユーザーの回答：** 実行ファイル

• **正解と解説：** **実行ファイル** です。

**質問10：Web版Claude Codeのタスクをローカルのターミナルに引き継ぐためのボタンの名前は何ですか？**

• **ユーザーの回答：** &

• **正解と解説：** 正解は**「Open in CLI」ボタン**です。

**質問11：C言語において「文字列」は何の配列ですか？また、末尾に自動挿入される特別な文字を何と呼びますか？**

• **ユーザーの回答：** chara の配列 / セミコロン

• **正解と解説：** 前半の「**char（文字）の配列**」は正解です。後半の正解は**ヌル終端文字 (\0)** です
