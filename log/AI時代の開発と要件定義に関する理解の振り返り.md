### **ソースに基づいた補足とまとめ**

[[2026-01-03T221944+0900_データ_2025年、AI時代の要件定義について考える|AI時代の要件定義]] では、人間の「決断」と「責任」の重要性が強調されています：

- **人間の「意志」と「責任」**: AIは過去のデータから効率的な案を出せますが、未来への「意志」は持たず、結果への責任も取れません。そのため、**「何を作らないか」を棄却する勇気**が、リソースを無駄にしないための防波堤となります。
- **Claude Codeの戦略的活用**: [[2026-01-03T220846+0900_データ_Claude Code Best Practices|Best Practices]] で紹介されているように、実装前の**「計画モード（Plan Mode）」**で人間が承認プロセスを挟むことや、**CLAUDE.md**にプロジェクトの記憶を蓄積すること、そして**複数のAIに「作成」と「検証」の役割を分担させること**が、開発効率と品質を両立させる鍵です。

今回のやり取りを通じて、[[2026-01-03T221944+0900_データ_2025年、AI時代の要件定義について考える|要件定義の本質]] と [[2026-01-03T220846+0900_データ_Claude Code Best Practices|Claude Code の実践的な活用法]] を組み合わせることで、AIを単なる「自動生成ツール」としてではなく、**「人間が最終判断を下すための強力な支援エージェント」として位置づける、AI時代の開発の核心部分を整理することができました。
